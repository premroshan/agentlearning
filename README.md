# Agentlearning

## Agents

### Why agents
Agents is similar to interactively working with an LLM to get optimised answer

Agents are software loops that iteratively work with an LLM to get required answer

### Multi agents

Helps to have specialised agents for each tasks. Each agents is specialised and more customisable

### What are tools

They are tools or skills that an LLM can use to make its output more relevant. Usually orchestrated by agent loops or custom apps / UI extension

## Agent framework - CrewAI

CrewAI is a multi AI agent system. 

Eg: Create agents to research and write an article
